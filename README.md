# ProjectOctane

## ANTES DE EJECUTAR START_DB.PY
- Crear una base de datos en postgresql
- Crear un archivo en la raiz del proyecto llamado .env
- Configurar las variables de entorno: DB_NAME, DB_HOST, DB_PASSWORD, KEY y DB_USER

## COMANDOS
- pip install psycopg2-binary
- pip install discord.py
- pip install dotenv
